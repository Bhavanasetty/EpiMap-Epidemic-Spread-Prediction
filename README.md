# EpiMap: Real-Time Epidemic Spread Prediction Framework

## Overview

EpiMap is a research-driven framework designed for real-time mapping and prediction of epidemic spread using data-driven and machine learning techniques. The system analyzes infection patterns and forecasts disease progression to support decision-making in public health systems.

This work has been accepted and presented at an IEEE international conference and is indexed in Scopus.


## Publication

* Title: EpiMap: A Comprehensive Framework for Real-Time Mapping and Prediction of Epidemic Spread
* Conference: IEEE ICCAMS 2025
* Indexing: Scopus Indexed

Note: Due to IEEE copyright restrictions, the full paper cannot be shared publicly. A summary of the work is provided in this repository.
